# Welcom to fast-alchemy

This project was first made to provide some tools to use Fastapi with SQLAlchemy with ease.

## Installation

Installation using pip:
```shell
pip install fast_alchemy
```

Or with poetry:
```shell
poetry add fast_alchemy
```
